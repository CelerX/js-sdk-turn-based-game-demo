# CelerX JS SDK demo for turn-based game: Gomoku

This is a simple demo for integerating a turn-based HTML5 game Gomoku with CelerX JS SDK. The game also enabled smart contract support.

## Start Project

> - run `npm install`
> - download `cocos creator` from <https://www.cocos.com/creator>
> - open the project in `cocos creator`

## Build Project

> - FYI <https://docs.cocos.com/creator/manual/en/publish/publish-web.html>
